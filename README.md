## 🟢 PHASE 1: SMALL PROJECTS (10 Apps)

1. Smart Calculator: Standard math operations plus memory recall (M+, M-), history log, backspace and some mathematical functions, keyboard support.

2. Dynamic Pomodoro Timer: Visual countdown clock with custom interval settings (Work, Short Break, Long Break), browser audio alerts, and tab title time updates.

3. Interactive Markdown Previewer: Split-screen editor where text typed into a textarea renders instantly as HTML markdown on the right side using a clean parsing library.

4. Local Storage Kanban Board: A simple 3-column drag-and-drop board (Todo, In Progress, Done) that persists your items locally in the browser even after a page refresh.

5. Real-time Currency Converter: Connects to a public free API to fetch live exchange rates and calculate dynamic conversions with search-as-you-type filtering.

6. Password Generator & Strength Meter: Generates customized secure passwords while evaluating complexity (entropy) using structural rules, featuring a "Copy to Clipboard" button.

7. Expense Tracker with Charting: Form to add expenses by category (Food, Rent) that visualizes data using a simple client-side SVG bar graph or pie chart.

8. GitHub Profile Finder: Fetch and display profile information, top repositories, and activity metrics using the GitHub Public API with clean error handling for missing users.

9. Flashcard Learning App: Digital quiz flashcards with a flipping visual effect, categorization, and a progress tracking indicator.

10. Custom HTML5 Video Player: Build visual controls from scratch for play/pause, scrub bars, custom volume sliders, and playback speed adjustments without using default browser UI wrappers.
